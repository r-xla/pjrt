# Convert to PJRT Client

Convert a platform name to a PJRT client or verify that an object is
already a client.

## Usage

``` r
as_pjrt_client(x)
```

## Arguments

- x:

  (`PJRTClient` \| `character(1)`)  
  Either a PJRT client object or a platform name (e.g., "cpu", "cuda",
  "metal").

## Value

`PJRTClient`
