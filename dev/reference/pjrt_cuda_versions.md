# Recommended CUDA Library Versions

A named list of the recommended library versions for CUDA support. The
PJRT CUDA plugin was built with these versions.

## Usage

``` r
pjrt_cuda_versions
```

## Format

A named list with character elements:

- cuda:

  CUDA version

- cudnn:

  cuDNN version

- nvshmem:

  NVSHMEM version
